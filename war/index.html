Automatically changing Bitcoin tipping addresses

Enter your Bitcoin addresses or BIP 32 Extended Public Key. You will receive code that you can add to any webpage, forum post, or Reddit post. This code will return you an address with no balance that you entered or a newly derived address from the xpub. This allows you to accept donations without worrying about address reuse.

The source code for this website is available at https://github.com/achow101/bittipaddr
